## Contributing

The code in this repository is provided only for didactical purposes. As such, it is tied to an article and I cannot updated it without also updating the article.

For this reason, please don't send any pull requests. If you want to contribute, I would really appreciate if you can [link to the original article](https://matteomanferdini.com/swiftui-asyncimage/) from your blog or share it on forums and social media. Thanks.
